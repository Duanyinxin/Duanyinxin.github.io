个人博客搭建（Windows）

0x01安装node.js

下载地址https://nodejs.org/en/

检查版本

win+R然后输入cmd

1、node -v

2、npm -v

0x02安装cnpm

npm install -g cnpm

国内速度较慢可以使用淘宝镜像源

（npm install -g cnpm –registry=http://registry.npm.taobao.org）

检查版本

cnpm -v

0x03安装hexo框架

cnpm install -g hexo-cli

检查版本

hexo -v

0x04创建blog文件夹

mkdir blog

默认建立在C:\Users\username文件夹下即命令行提示的路径下

根据自己需要可建立到自己想要的目录下先跳转盘符（D:）再输入路径（cd 路径）

0x05 生成blog

hexo init

Windows需要线安装git

git下载网址https://git-scm.com/downloads

选择对应版本

安装后再次打开cmd窗口转换到对应路径再次输入hexo init

遇到问题：

1、 OpenSSL SSL_read: Connection was reset, errno 10054

解决方案1、再试一次

​ 2、git config –global http.sslVerify “false”

2、Failed to connect to github.com port 443: Timed out

解决方案：取消代理git config –global –unset http.proxy

git config –global –unset https.proxy

0x06启动blog

hexo s

打开浏览器搜索localhost:4000

Ctrl + C关闭

0x07新建一篇博客

hexo n “我的第一篇博客文章”

hexo clean

hexo g

hexo s

0x08部署到GitHub

1、登录GitHub

2、新建仓库（用户名.github.io）

3、命令行输入cnpm install –save hexo-deployer-git安装git部署插件

0x09配置_config.yml

修改最后几行为

0x10推送

hexo d

注意git需要

git config –global user.email “you@example.com“ git config –global user.name “Your Name”

0x11检查是否成功

浏览器打开（YourGitHubname.github.io）
